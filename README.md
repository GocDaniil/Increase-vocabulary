# Increase-vocabulary
## Увеличьте свой словарный запас
Данное Android приложение дает вам возможность выучить слова на определенную «частоту произношения» и «тему»
Словарь содержит более 16 тысяч слов - это сжатый вариант словаря из используемых слов в русской речи
